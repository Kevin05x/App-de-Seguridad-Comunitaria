# 🚨 Alerta Prevención Comunitaria

> Una solución integral para la seguridad ciudadana y respuesta ante emergencias.

---

## 📖 Resumen del Proyecto
Esta aplicación permite a las comunidades organizarse proactivamente contra el delito. Resuelve la falta de comunicación inmediata mediante un **Botón de Pánico (SOS)** geolocalizado y un **Muro de Seguridad** colaborativo.

## 🛠️ Tecnologías Principales

| Capa | Tecnología |
| :--- | :--- |
| **Lenguaje** | Java (Android SDK) |
| **Base de Datos Local** | Room (SQLite) |
| **Backend / Nube** | Firebase (Firestore, Auth, Storage, Messaging) |
| **Geolocalización** | Google Maps API & Fused Location |
| **Seguridad** | jBCrypt (Hashing) |

## 👨‍💻 Liderazgo y Desarrollo (Mi Rol)
Como **Product Owner y Desarrollador Principal**, lideré el ciclo completo de vida del software:

* **Ideación:** Recolección de ideas y necesidades comunitarias.
* **Arquitectura:** Diseño del esquema de datos y estrategia *Offline-First*.
* **Full-Stack:** Desarrollo de lógica de negocio, integración de APIs y UI/UX (Material Design).

## 🚀 Funcionalidades Clave
* **Modo Offline-First:** Los reportes se guardan localmente si no hay red.
* **Alertas en Tiempo Real:** Notificaciones instantáneas a todos los vecinos.
* **Evidencia Multimedia:** Posibilidad de adjuntar fotos a los reportes de seguridad.
